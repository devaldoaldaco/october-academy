Introducción a HTML semántico

5 de octubre del 2025

¿Qué es HTML semántico?

HTML5 introduce nuevas etiquetas semánticas que dan significado al contenido de nuestras páginas web. Estas etiquetas ayudan a los navegadores y motores de búsqueda a entender mejor la estructura del documento.

Beneficios principales

El uso de etiquetas semánticas mejora la accesibilidad, el SEO y la mantenibilidad del código. Facilita el trabajo de los lectores de pantalla y hace que tu código sea más legible para otros desarrolladores.

Etiquetas más usadas

Entre las etiquetas semánticas más importantes encontramos: header, nav, main, article, section, aside, footer, figure y figcaption. Cada una tiene un propósito específico en la estructura del documento.
